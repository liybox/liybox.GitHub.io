python的基础教程——python的准备工作

发表于 2019-08-15 | 更新于: 2019-08-15 | 分类于 python |

对于我们菜鸟来说，开发环境的选择非常重要，一般Python的开发环境选择Linux。
Python2.x与Python3.x有着很大程度上的不兼容性，但是毕竟随着Python的不断发展，Python3.x才是未来的主流，所以我们选择安装Python3.x的环境。

在Windows上安装Python

  1. 从 Python 的官方网站下载对应的Python 版本，官方地址：https://www.python.org/downloads/![在这里插入图片描述](https://img-blog.csdnimg.cn/20190815094940194.png)
    最新的版本已经更新到3.7.4，下载即可。
  2. 运行下载的恶exe安装包：

    注意选择Add Python 3.6 to PATH 然后点 Install Now 可完成安装。默认会安装到 C:\Python36 目录下。
  3. 打开命令提示符窗口（方法是点击“开始”-“运行”- 输入：“cmd”），敲入 Python 后，会出现两种情况：
    一：

    出现这个说明安装完成，且路径正确，你看到提示符 >>> 就表示我们已经在 Python 交互式环境中了，可以输入任何Python 代码，回车后会立刻得到执行结果。现在，关掉命令行窗口，就可以退出 Python 交互式环境。
    二：如果出现：’python‘ 不是内部或外部命令，也不是可运行的程序
    这是因为 Windows 会根据一个 Path 的环境变量设定的路径去查找 Python.exe，如果没找到，就会报错。如果在安装时漏掉了勾选 Add Python 3.x to PATH ，那就要手动把 Python.exe 所在的路径添加到 Path 中。

如果你不知道怎么修改环境变量，建议把 Python 安装程序重新运行一遍，务必记得勾上Add Python 3.x to PATH。

在Mac上安装Python


如果你正在使用 Mac，系统是 OS X 10.8～10.10，那么系统自带的 Python 版本是 2.7，需要安装最新的 Python 3.x。
第一步：
方法一：下载安装

从 Python 官网下载 Python 安装程序
然后只需要一直点击继续就可以安装成功了。

方法二： Homebrew 安装

如果安装了 Homebrew，直接通过命令 brew install Python3 安装即可。

第二步:
如果不放心，可以再检查一下。操作方法是打开终端，输入 Python3（不是输入 Python 3 ，也不是 Python )

得到这样的结果，就说明安装成功了。
