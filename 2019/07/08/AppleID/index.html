注册国外AppleID，你学会了么？

发表于 2019-07-08 | 更新于: 2019-07-14 |

在苹果的AppStore，很多软件在国区找不到，或者说在国区被下架了。

我们登录国外Apple ID，往往能发现，原来换个区就能下载了。

我的一个注册办法：

一个很小小小小众的办法…

这一次，更简单了，只需三步！

废话不多说，看下面教程。

一、注册国内Apple ID

打开：https://appleid.apple.com/

翻到页面底部，创建新的Apple ID，注册一个中国区的Apple ID就行了

注册国区Apple ID很简单，填写你用于注册的邮箱、手机号、安全提示问题等信息。

我在这里就不截图演示了。

注意自己记录一下注册时填写的安全提示问题

二、登录AppStore一次

刚刚注册好的Apple ID，请先在你的IOS设备的AppStore登陆一次

1.点击「Review/检查」

因为你是第一次登录Appstore，会有如下提示。

（你们看到的应该是中文界面）

2.选择付款方式为「无」

选择付款方式为无，然后点击右上角的「Next/下一步」

三、切换国家

终于来到最后一步了

1.打开Apple ID网页

还是打开：https://appleid.apple.com/

然后登陆刚刚注册的帐号，点击右上角的「编辑」

2.改国家为菲律宾

记得，切换国家选择「菲律宾」

会有下图提示，请点击「继续更新」

3.为所欲为

然后你就可以发现

你可以直接选择付款方式为无！

最后，剩下的信息，你就可以随意填写啦

在标示了位数的地方，填写对位数就行

数字可以随便写

填完后，记得点页面右上角的保存

然后

你就拥有一个菲律宾的Apple ID啦！

4.最后说一下缺点

用这个办法，暂时只能转到菲律宾，转其他区的话，必须填写国外付款方式。

知乎评论中网友反馈得知同样的方法还可以改「香港」

地址什么的可以乱填

唯一需要注意的地方是，最后填写手机的地方：852-XXXXXXX（乱填7位数）

但是菲律宾区的Apple ID，也已经可以下载很多国区被下架的App了。比如Ⅴρn什么的。

如果需要注册其他区的AppleID，还请关注后续的教程：简单十步注册一个国外Apple ID

建议大家都注册一个备用。

有一个国外Apple ID，某些时候会方便很多
